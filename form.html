What is Form tag in HTML?

The <form> tag is used for creating a form for user input. A form can contain textfields, checkboxes, radio-buttons & more.
SYNTAX OF FORM TAG < form > < /form >

What is Input tag in HTML?

The <input> tag specifies an input field where the user can enter data. The <input> element is the most important
form element. The <input> element can be displayed in several ways, depending on the type attribute.
SYNTAX OF INPUT TAG <input />

What is Type Attritubes in HTML Input Attributes?

The <input> type attribute specifies the type of <input> element to display. If the type attribute is not specified,
the default type is "text".
SYNTAX OF TYPE ATTRIBUTES <input type=""/>

What is Password Type Attritubes in HTML Input Tag?

The <input> elements of type password provide a way for the user to securely enter a password. The element is presented
as a one-line plain text editor control in which the text is obscured so that it cannot be read, usually by replacing
each character with a symbol such as the asterisk ("*") or a dot ("â€¢").
SYNTAX OF PASSWORD TYPE ATTRIBUTES <input type="password"/>

What is Radio Type Attritubes in HTML Input Tag?

The <input> radio button state represents a type attribute whose value is "radio" represents a selection of one item from
a list of items (a radio button).
SYNTAX OF RADIO TYPE ATTRIBUTES <input type="radio"/>

What is CheckBox Type Attritubes in HTML Input Tag?

The <input> elements of type checkbox are rendered by default as boxes that are checked (ticked) when activated,
like you might see in an official government paper form. A checkbox allows you to select single values for submission
in a form (or not).
SYNTAX OF CHECKBOX TYPE ATTRIBUTES <input type="checkbox"/>

What is Email Type Attritubes in HTML Input Tag?

The <input> type value is automatically validated to ensure it is a properly formatted e-mail address. To define an e-mail
field that allows multiple e-mail addresses, add the "multiple" attribute.
SYNTAX OF EMAIL TYPE ATTRIBUTES <input type="email"/>

What is Search Type Attritubes in HTML Input Tag?

The <input> element, having the "search" value in its type attribute, represents a one line string field for search
purposes. The control associated to this field is a text box that allows users to edit a single line of plain text.
SYNTAX OF EMAIL TYPE ATTRIBUTES <input type="search"/>

What is Range Type Attritubes in HTML Input Tag?

The <input> type defines a control for entering a number whose exact value is not important (like a slider control).
Default range is 0 to 100.
SYNTAX OF RANGE TYPE ATTRIBUTES <input type="range"/>

What is Time Type Attritubes in HTML Input Tag?

The <input> element, having the "time" value in its type attribute, represents a field for a time input. In modern browsers
time fields are usually represented by controls that enable users to change its value in a graphical way, instead of
having to input it directly as a string.
SYNTAX OF TIME TYPE ATTRIBUTES <input type="time"/>

What is Month Type Attritubes in HTML Input Tag?

The <input>elements of type month create input fields that let the user enter a month and year allowing a month
and year to be easily entered. The value is a string whose value is in the format " YYYY-MM ", where YYYY is the
four-digit year and MM is the month number.
SYNTAX OF MONTH TYPE ATTRIBUTES <input type="month"/>

What is Week Type Attritubes in HTML Input Tag?

The <input>element, having the "week" value in its type attribute, represents a field for a week input.
SYNTAX OF WEEK TYPE ATTRIBUTES <input type="week"/>

What is Reset Type Attritubes in HTML Input Tag?

The <input> reset button allows users to basically clear their web form. It wipes values from all fields by "resetting"
the form to its default appearance.
SYNTAX OF RESET TYPE ATTRIBUTES <input type="reset"/>

What is Submit Type Attritubes in HTML Input Tag?

The <input> type defines a submit button which submits all form values to a form-handler. The form-handler is typically
a server page with a script for processing the input data.
SYNTAX OF SUBMIT TYPE ATTRIBUTES <input type="submit"/>

What is File Type Attritubes in HTML Input Tag?

The <input> element, having the "file" value in its type attribute, represents a control to select a list of one or more
files to be uploaded to the server.
SYNTAX OF FILE TYPE ATTRIBUTES <input type="file"/>

What is Color Type Attritubes in HTML Input Tag?

The <input> elements of type color provide a user interface element that lets a user specify a color, either by using a
visual color picker interface or by entering the color into a text field in #rrggbb hexadecimal format.
SYNTAX OF COLOR TYPE ATTRIBUTES <input type="color"/>
