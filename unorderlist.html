What is HTML Unordered List?

HTML Unordered List displays elements in bulleted format.
Between the unordered list tag, the tag list item< li > is used to represent an item in a list.
Name Syntax Uses
Unordered list <ul> </ul> The HTML ul tag is used for creating the bulleted list, those where the order of the items is arbitrary.

Example:

< ul >

<li> Apple <li>
<li> Mango <li>
<li> Banana <li>

</ ul >

Result:

• Apple
• Mango
• Banana

Unordered List Attributes:


Name Syntax Description Output
Disc < ul type=disc > </ ul > Sets the list item marker to a bullet(default) • Disc
Circle < ul type=circle > </ ul > Sets the list item marker to a circle ○ Circle
Square < ul type=square > </ ul > Sets the list item marker to a square □ Square

Example:

< ul type=disc >
< li > Tomato </li >
</ ul >

Result:

• Tomato
