What is strong tag in HTML?

The <strong> tag is used to define text with strong importance. The content inside is typically displayed in bold.
SYNTAX OF STRONG TAG < Strong > < /Strong >

What is bold tag in HTML?

The <b> tag is used to create a 'b' element, which represents bold text in an HTML document.
SYNTAX OF BOLD TAG < b > < /b >

What is Emphasized tag in HTML?

The <em> tag is used to define emphasized text. The content inside is typically displayed in italic.
SYNTAX OF EMPHASIZED TAG < em > < /em >

What is Italic tag in HTML?

The <i> tag inside is typically displayed in italic.
SYNTAX OF ITALIC TAG < i > < /i >

What is SuperScript tag in HTML?

The <sup> tag specifies inline text which is to be displayed as superscript for solely typographical reasons.
Superscripts are usually rendered with a raised baseline using smaller text.
SYNTAX OF SUPERSCRIPT TAG < sup > < /sup >

What is SubsScript tag in HTML?

The <sub> tag  defines subscript text. Subscript text appears half a character below the normal line, and is sometimes
rendered in a smaller font. Subscript text can be used for chemical formulas, like H2O.
SYNTAX OF SUBSCRIPT TAG < sub > < /sub >

What is Horizontal Rule tag in HTML?

The <hr> tag  element is most often displayed as a horizontal rule that is used to separate content.
SYNTAX OF HORIZONTAL RULE TAG < hr/ >

What is Break tag in HTML?

The <br> tag element produces a line break in text (carriage-return). It is useful for writing a poem
or an address, where the division of lines is significant.
SYNTAX OF BREAK TAG < br/ >

What is Underline tag in HTML?

The <u> tag  inside is typically displayed with an underline.
SYNTAX OF UNDERLINE TAG < u> < /u >

What is Preformatted tag in HTML?

The <pre> tag  element represents preformatted text which is to be presented exactly as written in the HTML file
SYNTAX OF PREFORMATTED TAG < pre> < /pre >

What is Big tag in HTML?

The <big> tag  is used to make the text one size bigger (ie: from small to medium, medium to large, large to x-large).
SYNTAX OF BIG TAG < big> < /big >

What is Small tag in HTML?

The <small> tag is used to make the text one size smaller (ie: from x-large to large, large to medium, medium to small).
SYNTAX OF SMALL TAG < small> < /small >
