Heading

HTML headings are titles or subtitles that you want to display on a webpage.

HTML headings are denoted by <h1> to <h6> tags.

In HTML, <h1> defines the most important heading while, <h6> defines the least important heading.

Name Syntax Uses of Headings
Heading <h1 to h6> <h1> headings should be used for main headings, followed by <h2> headings, then the less important <h3>, and so on.

Attribute

Name Value Description
align left, right, center, justify "Obsolete":Specifies the horizontal alignment of text within the paragraph.

Examples

<h1> The Quick Brown Fox Jumps Over The Lazy Dog. <h1>

<h2> The Quick Brown Fox Jumps Over The Lazy Dog. <h2>

<h3> The Quick Brown Fox Jumps Over The Lazy Dog. <h3>

<h4> The Quick Brown Fox Jumps Over The Lazy Dog. <h4>

<h5> The Quick Brown Fox Jumps Over The Lazy Dog. <h5>
<h5> The Quick Brown Fox Jumps Over The Lazy Dog. <h6>

Paragraph

Paragraphs are usually represented in visual media as blocks of text separated from adjacent blocks by blank lines and/or first-line indentation.

Name Syntax Uses of Headings
Paragraph <p> The <p> tag defines as paragraph. It is in pair form.

Attribute

Name Value Description
align left, right, center, justify "Obsolete":Specifies the horizontal alignment of text within the paragraph.
