What is an Order List Tag?

The < ol > tag defines an ordered list. An ordered list can be numerical or alphabetical.

Name Syntax Use
Order List < ol > Order is used to arrange the things in a specific order. This list can be presented in numerical way or alphabetical way

What is a List Tag?

 The < li > tag is used to define each list  item

Name Syntax Use
List Tag < li > List tag is used to define the each listed item.

Example:

 < ol >
< li >Coffee< /li >
< li >World< li >
< li >Life < li >
< /ol >

Result:

1 Coffee
2 World
3 Life
