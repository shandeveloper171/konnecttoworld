What is HTML?

HTML stand for Hyper Text Markup Language.

It is used to make the structure of website.

You can make your own website with HTML

What is HTML tags?

HTML tags are the hidden keywords within a web page that define how your web browser must format and display 
the content.Most tags must have two parts, an opening and a closing part. HTML tags contain three main parts:
opening tag, content & closing tag. Some of HTML tags are used as single not in pair
EXAMPLE OF TAGS < h1 > < /h1 > | < br />

What is Attributes in HTML tags?

HTML attributes are special words used inside the opening tag to control the element's behaviour.
HTML attributes are a modifier of an HTML element type.
EXAMPLE OF ATTRIBUTES width = " "

What is Body tags in HTML?
