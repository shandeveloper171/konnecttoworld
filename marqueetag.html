Marquee Tag

The <marquee> tag is a container tag of HTML.

It is implemented for creating scrollable text or images within a web page from either left to right or vice versa, or top to bottom or vice versa.

But this tag has been Deprecated in the new version of HTML, i.e., HTML 5.

Name Syntax Uses of Headings
Marquee Tag <marquee> The HTML <marquee> element is used to insert a scrolling area of text.

Attributes

Names Descriptions
behavior Sets how the text is scrolled within the marquee. Possible values are scroll, slide and alternate.
bgcolor Sets the background color through color name or hexadecimal value.
direction Sets the direction of the scrolling within the marquee. Possible values are left, right, up and down.
height Sets the height in pixels or percentage value.
loop Sets the number of times the marquee will scroll.
scrollamount Sets the amount of scrolling at each interval in pixels. The default value is 6.
scrolldelay Provides a feature whose value will be used for delaying among each jump.
