What is Table Tag?

A simple HTML table, containing columns and rows:

The < table > tag defines an HTML table:
An HTML table consists of one <table> element and one or more <tr>, <th>, and <td> elements.
These elements <tr>, <th>, and <td> are further used in table tag to becomes a proper table.

Syntax Description
<tr> </tr> This element defines a table row
<th> </th> This element defines a table header
<td> </td> This element defines a table cell

Example:

< table border=3 >
< tr >
< th > Sr.No </ th >
< th > Name </ th >
< th > Position </ th >
< /tr >

< tr >
< td > 1 </ td >
< td > Ali </ td >
< td > 1st </ td >
< /tr >
< /table >

Result:

Sr.No Name Position
1 Ali 1st

Table Tag Attributes:

  • Border
  • Cellpadding
  • Cellspacing

Attributes Description
Border The HTML < table > border Attribute is used to specify the border of a table. It sets the border around the table cells
Cellpadding It represents the distance between cell borders
Cellspacing This attribute defines space between table cells

Example:

< table border=3 Cellpadding=10% Cellspacing=10% >
< tr >
< th > Sr.No < /th >
< th > Name < /th >
< th > Position < /th >
< /tr >
< tr >
< td > 1 < /td >
< td > Ali < /td >
< td > 1st < /td >
< /tr >

Result:

Sr.No Name Position
1 Ali 1st

Tr & Td Attributes:

Two attributes which used in both tr and td is following as:
  • Colspan
  • Rowspan

Attributes Description
Colspan The colspan attribute defines the number of columns a cell should span.
Rowspan The rowspan attribute specifies the number of rows a cell should span.

Example:

Colspan:

< table border="3" >
< tr >
< td colspan="50%" >Ali< /td >
< td >Hamza< /td >
< /tr >
< /table >

Result:

Ali Hamza

Rowspan:

< table border="3" >
< tr >
< td rowspan="50%" >Ali< /td >
< td >Hamza< /td >
< /tr >
< tr >
< td >Hadi< /td >
< /tr >
< /table >

Result:

Ali Hamza
Hadi
